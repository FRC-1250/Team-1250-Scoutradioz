# 2026 Scoutradioz Pit and Match scouting forms

***Important***\
Be sure to upload the 3 images to your org, or remove them from the code, to validate the form!

## Pit form
- Records drive base type, length, and width
- Hopper dimensions to calculate fuel capacity
- Auto routines
- Team friendliness :)

## Match form
- Uses a robot's hopper dimensions from pit scouting to calculate an estimated fuel capacity (60% packing efficiency)
- Scouts count Payloads (full hopper), half Payloads, and quarter Payloads scored per robot
- Manual fuel counter included for stray shots and refinement
- Robot fuel capacity is set to 15 fuel if they have not been pit scouted
- Records passing and playing defense in the Neutral Zone and Opponent's Zone
- Pulls Auto Tower Points and Endgame Tower Points from The Blue Alliance
